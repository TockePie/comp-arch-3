# Лабораторна робота №4.2

## Відомості

### Тема

Робота з перериваннями (LEDs and Button)

### Мета

Ознайомлення з принципами роботи контролера зовнішніх переривань (EXTI) у мікроконтролерах серії STM32F4. Набуття навичок налаштування периферії через STM32CubeMX та написання обробників переривань для взаємодії з апаратними кнопками та світлодіодами.

### Обладнання та програмне забезпечення

- ОС: Ubuntu 20.04 (Virtual Machine)
- IDE: STM32CubeIDE v1.11.2
- Емулятор: xPack QEMU Arm v7.2.0-1

### Варіант

```bash
> console.log((3210 % 5))
0
```

0. Натискання на кнопку запалює або гасить відразу всі світлодіоди.

## Хід роботи

### Опис підключення периферії

Згідно зі схемою плати та налаштуваннями у проекті:

- **PA0 (User Button):** налаштовано як джерело зовнішнього переривання `GPIO_EXTI0`.
- **PD12, PD13, PD14, PD15:** налаштовані як цифрові виходи (`GPIO_Output`) для керування світлодіодами зеленого, помаранчевого, червоного та синього кольорів відповідно.

### Виконання роботи та код програми

Для реалізації **Варіанту 0** (синхронне перемикання всіх світлодіодів) було модифіковано файл `stm32f4xx_it.c`. Використано глобальну змінну `state` для збереження поточного стану системи.

**Обробник переривання:**

```c
void EXTI0_IRQHandler(void)
{
  // Визначаємо маску для всіх чотирьох світлодіодів
  uint16_t all_leds = GPIO_PIN_12 | GPIO_PIN_13 | GPIO_PIN_14 | GPIO_PIN_15;

  if (!state) {
      // Якщо стан 0 — вмикаємо всі світлодіоди синхронно
      HAL_GPIO_WritePin(GPIOD, all_leds, GPIO_PIN_SET);
  } else {
      // Якщо стан 1 — вимикаємо всі світлодіоди
      HAL_GPIO_WritePin(GPIOD, all_leds, GPIO_PIN_RESET);
  }

  // Інвертуємо стан для наступного натискання
  state = !state;

  // Очищення прапора переривання через HAL-драйвер
  HAL_GPIO_EXTI_IRQHandler(GPIO_PIN_0);
}
```

### Результати моделювання

Тестування проводилося в емуляторі QEMU. При натисканні на синю кнопку `User` у консолі термінала фіксується одночасна зміна стану всіх чотирьох LED-індикаторів.

> _Робота програми в QEMU. У логах чітко видно послідовне вмикання/вимикання всіх кольорів (green, orange, red, blue) при взаємодії з кнопкою._

### Висновки

У ході роботи було успішно реалізовано алгоритм керування світлодіодами за допомогою зовнішніх переривань. Використання механізму **EXTI** дозволяє миттєво реагувати на дії користувача, не перевантажуючи основний цикл програми `while(1)` постійним опитуванням стану піна. Завдання варіанту 0 виконано повністю.
